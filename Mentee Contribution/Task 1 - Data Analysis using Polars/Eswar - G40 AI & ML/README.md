# 🔥 Wildfire Risk Intelligence Engine

## Overview
A Streamlit web app that analyzes NASA VIIRS SNPP fire archive data to detect and visualize wildfire risk patterns globally.

## Dataset
NASA FIRMS VIIRS SNPP Fire Archive — loaded via Google Drive

## Tech Stack
- Python
- Streamlit
- Pandas
- NumPy
- Matplotlib
- gdown

## Features
- Monthly fire detection trends
- Brightness temperature distribution
- Day vs Night fire breakdown
- Confidence level analysis
- FRP distribution
- Geographic hotspot map
- Top hotspot grid cells table
- Downloadable filtered data
